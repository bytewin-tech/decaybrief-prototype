# DecayBrief Prototype

Frontend prototype for **DecayBrief**, a micro SaaS concept that helps creators and SEO teams identify decaying content and generate refresh briefs.

## What is included

- Landing page hero and value proposition
- CSV import / opportunity dashboard screen
- Article detail screen with decay score and recommended actions
- Generated refresh brief / export screen
- Realistic sample content and monetizable positioning

## Stack

- Self-contained HTML, CSS, and JavaScript
- No build step required
- Deployable as a static site on Netlify

## Local usage

Open `index.html` directly in a browser, or serve the folder with any static server.

Example:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deployment notes

- Git repo initialized and pushed to GitHub before deployment
- Production site deployed to Netlify from the local folder
- Verification performed with HTTP checks and browser console inspection
